# Aura Weather

A modern, responsive, and visually engaging weather web application that provides real-time data and immersive animated backgrounds.

![Aura Weather Preview](https://github.com/dan-delion-source/Simple-Weather-Forcast/raw/main/preview_recording.webp)

## Features

- **Real-time Weather Data**: Comprehensive metrics including temperature, humidity, wind speed, UV index, and Air Quality Index (AQI).
- **Global Search with Autocomplete**: Quickly find any location worldwide with debounced search suggestions and a high-contrast UI for readability.
- **Dynamic Animated Backgrounds**: Immersive canvas-powered backgrounds that react to current weather (Clear, Cloudy, Rain, Snow).
- **Glassmorphism Design**: Premium UI with frosted glass effects, smooth transitions, and a clean minimalist layout.
- **5-Day Forecast**: Detailed upcoming weather patterns and temperatures.
- **Responsive Design**: Optimized for desktops, tablets, and mobile devices.

## Technology Stack

- **HTML5**: Semantic structure.
- **CSS3**: Modern layout (Flexbox/Grid), Glassmorphism, and custom animations.
- **JavaScript (ES6+)**: Functional logic and API integration.
- **Canvas API**: Custom particle-based weather animation engine.
- **Open-Meteo API**: High-fidelity, real-time weather and environmental data.

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/dan-delion-source/Simple-Weather-Forcast.git
   ```
2. Open `index.html` in your favorite browser.
3. Search for any city to experience Aura Weather!

## API Usage

This project uses the [Open-Meteo API](https://open-meteo.com/), which provides free access for non-commercial use without requiring an API key.
